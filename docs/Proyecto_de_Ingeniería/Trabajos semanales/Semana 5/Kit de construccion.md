

Para poner en práctica el uso de la cortadora láser, elaboramos un kit de construcción mediante el diseño de las plantillas de las piezas, las cuales fueron posteriormente cortadas en la máquina para su ensamblaje final.

Para este proyecto diseñé un tocador. Comencé realizando la plantilla en SolidWorks, utilizando la herramienta de variables globales para definir las dimensiones de las ranuras y asegurar el correcto ensamblaje de las piezas.

Para este proyecto utilicé madera balsa, por lo que fue necesario adaptar las medidas de las ranuras conforme a las propiedades del material. Por ejemplo, el grosor de las ranuras se ajustó a 6 mm para asegurar un ensamblaje correcto. De la misma manera, este material influyó en la configuración de la máquina láser, ya que, al ser delgado, no fue necesaria una potencia alta.
