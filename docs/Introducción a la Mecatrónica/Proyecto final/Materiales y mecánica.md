# Materiales y Mecánica
---

Como proyecto final se desarrolló una plataforma móvil capaz de centrar y mantener una pelota en equilibrio mediante el uso de servomotores o motores lineales. El sistema responde automáticamente a los desplazamientos de la pelota, integrando conocimientos de electrónica, programación y mecánica para el desarrollo de un sistema de control funcional.

Para su contrucción se usaron los siguientes materiales: 

- 2x tablas de mdf de 40x40cm
- 4x motores lineales con 10 cm de carrera
- 4x soportes para motores lineales impresos en 3D y diseñados en SolidWorks
- 1x ESP32
- 2x Puentes H
- 4x soportes para la plataforma.


Se utilizaron dos tablas de MDF de 40 × 40 cm: una para la base, donde se montaron los motores, y otra para la plataforma superior. Ambas fueron pintadas de negro para mejorar el contraste y facilitar que la cámara detectara el color de la pelota.
